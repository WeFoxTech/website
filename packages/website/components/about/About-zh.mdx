
# 关于微狐科技
我们是一个组织，致力于为中小微企业和个人提供技术咨询和顾问服务，帮助企业降低研发成本（时间、资金），由创始人fox发起，目前团队成目员都是他身边的好朋友，有些是他帮一些创业团队担任技术顾问期间认识的。我们在特定技术领域都有着深入研究。我们从业经历覆盖了国内一线互联网大厂，目前大部分依然在自己所在的公司从事技术研发工作，也有一些开始了自己的创业之路。

# 中小微企业的痛点
顶部人才都往大厂靠拢、中小微企业对人才吸引力不足，人才资源不是很充沛，很容易被一些技术难点block住，早期没有丰富的架构经验，技术架构阻碍了市场的成长机遇，缺乏高效的研发流程，技术研发无法快速满足客户需求变化，要解决的上下游技术问题太多，无法专注于自己的核心业务...


# 技术解决方案
我们不仅仅提供技术咨询和顾问服务，我们还沉淀了许多通用的技术解决方案

- Cloud Native 云原生
  - k8s(Kubernetes)
  - Prometheus
  - Helm
  - ......

- 基于electron的桌面端封装方案
  - 基于typescript封装
  - 完全基于ipc的message机制抽象
  - 基于iframe 和web端保持一致，无须跟随web端一起发版
  - 支持win7、win8、win10、macOS、linux
  - 基于GitHub Actions的自动发版流水线

- 前端的加载优化
  - code splitting
  - bundle瘦身
  - 懒加载
  - tree shaking

- i18n 全球化解决方案（试用参考本站）
  - 体验超越国内各一线大厂各产品页
  - 无感切换、杜绝白屏闪烁、杜绝跳开用户当前浏览位置
  - 语言文件基于TypeScript，完美的编程体验和严格的校验机制
  - 支持next.js 最新9.3版本新出的SSG特性、也支持传统SSR

- 全屏幕适配方案
  - 一次开发适配多种屏幕大小
  - pc、mobile、微信、pad
  - 从iPhone SE到iPad Pro，只要你见过没见过的屏幕大小，我们都能完美适配
  - 通过改变浏览器窗口或者不同设备打开我们官网测试 [wefox.tech](https://wefox.tech)

- DevOps、 CI/CD
  - 基于GitHub Actions自动流水线
  - 基于Azure pipelines自动流水线
  - 基于AWS的云服务系统架构
