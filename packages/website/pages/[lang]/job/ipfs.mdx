如果你能解答下面两道题目其中一题，请联系我们，我们直接录用。如果能解答两道**薪资翻倍**。 可以借助[google](https://www.google.com/),[百度](https://www.baidu.com/), [必应](https://www.bing.com/)等各大搜索引擎，也可以求助身边的朋友。总之我们不限制你解题的方式。也不限制时间，但是也别太久哦，我们招满人之后就关闭职位了。

1. 把一个多页的 web 应用部署到[ipfs](https://ipfs.io/)网络

   - 通过普通 http 网关可以正常使用
   - 页面之间可以相互跳转
   - 子页面可以直接浏览器打开访问
   - 告知我们 webapp 的[CID](https://docs.ipfs.io/guides/concepts/cid/)。
   - 比如([`QmctDmAgS75nxUGiCaUzZodWBAq3MXAmEPmZ8EVCrkdnF1`](https://ipfs.io/ipfs/QmctDmAgS75nxUGiCaUzZodWBAq3MXAmEPmZ8EVCrkdnF1/event/))

2. 以矿工身份加入[filecoin](https://filecoin.io/)[测试网](https://stats.testnet.filecoin.io/)，告知我们你的矿工 id 比如(`t044499`)。

# ipfs、filecoin 技术研究员

## 工作内容

- [ipfs](https://ipfs.io/),[filecoin](https://filecoin.io/)生态技术研究

## 招聘需求

- 非在职状态
- 计算机相关专业专科或以上（如果你对 ipfs 非常了解，不受此限制）
- 有阅读英文文档，追溯源码解决问题的能力
- 熟悉 linux 操作系统使用，能阅读和编写简单的 bash 脚本

- 了解区块链的技术原理、有 Dapp 研发经历
- 了解[filecoin](https://filecoin.io/)项目，了解其中的共识原理

- 熟练 docker 的使用、有 k8s 使用经验
